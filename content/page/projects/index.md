## Distributed Task Queue

**Stack:** Go · Redis · PostgreSQL

A lightweight distributed task queue with at-least-once delivery guarantees, worker pools,
and a dead-letter queue for failed jobs. Built to handle high-throughput job scheduling
with minimal operational overhead.

[GitHub →](https://github.com/tannatsri)
